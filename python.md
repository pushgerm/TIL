    * 문자열 인덱싱과 슬라이싱  
        * 인덱싱 : 문자열 내 특정한 값을 뽑아냄  
        a = 'Life is too short'  
        a[3]은 e이다
        a[-1]은 n이다. 음수는 뒤에서부터 센다  
        * 슬라이싱 : 범위 내의 문자를 뽑아냄  
        a[0:3] : 0부터 3미만의 인덱스에 해당하는 문자를 슬라이싱

        val1 <= a < val2 에 해당하는 문자를 가져온다.
        앞번호를 생략 시 처음부터 가져오고 끝번호를 생략 시 끝까지 가져온다.  
        a = '20171218sunny'  
        year = a[:4]  
        month = a[4:6]  
        day = a[6:8]  
        weather = a[8:]  
        다음과 같이 문자열을 나눌 수 있다.

    * 문자열 포매팅 : 문자열 안에 어떤 값을 삽입하는 방법
        1. 숫자 바로 대입  
        ' I eat %d apples' % 3  
        -> I eat 3 apples  

        2. 문자열 바로 대입  
        ' I eat %s apples' % 'five'  
        -> I eat five apples  

        3. 숫자 값을 변수로 대입  
        a = 3  
        'I eat %d apples' % a  
        -> I eat 3 apples  

        4. 2개 이상의 값 넣기  
        number = 10  
        day = 2  
        'I ate %d apples in %d days' % (number, day) 

    * 문자열 포맷 코드  
    %s : 문자열(string)  
    %c : 문자 1개(character)  
    %d : 정수(integer)  
    %f : 부동소수(floating-point)  
    %o : 8진수  
    %x : 16진수  
    %% : Literal%(문자 %자체)  
    ** %s 는 어떤 형태의 값이든 변환해 넣을 수 있다.

    * 포맷 코드와 숫자 함께 사용하기
    
    1. 정렬과 공백  
    "%10s" % "hi"  
    ->           hi : 10칸의 공간에서 hi를 오른쪽으로 정렬하고 나머지는 공백    
    '%-10sjane" % "hi"  
    ->hi          jane : 10칸의 공간에서 hi를 왼쪽으로 정렬하고 공백

    "%0.4f" % 3.141592  
    -> 3.1415  
    정수부는 위 처럼 공백을 나타대고 소수점 이하 부분은 몇자리의 소수점을 나타낼지 정한다.

    * 리스트
        * 리스트는 어떻게 만들고 사용할까

        * 리스트의 인덱싱과 슬라이싱  
        a = [1, 2, 3, ['a', 'b', 'c']]  
        a[-1][0]  
        -> 'a'  
        요소 자체가 리스트일 경우 내부에서 한번 더 접근이 가능하다.  
          
        앞서 문자열의 슬라이싱과 같이 리스트를 슬라이싱 할 수 있다.  
        a = [1, 2, 3, ['a', 'b', 'c'], 4, 5]  
        a[2:5]  
        -> [3, ['a', 'b', 'c'], 4]  
        a[3][:2]  
        -> ['a', 'b']  
    * 리스트 연산자
        1. 더하기  
        리스트 요소들이 통합된다. 문자열에서의 더하기와 같다.  
        a = [1, 2, 3]  
        b = [4, 5, 6]  
        a + b  
        [1, 2, 3, 4, 5, 6]  
        같은 데이터 타입에서만 가능하며 데이터 타입이 다를시 에러가 난다.  

        2. 반복하기 *  
        a = [1, 2, 3]  
        a * 3  
        [1, 2, 3, 1, 2, 3, 1, 2, 3]  

        3. 리스트 수정, 변경, 삭제  
        a[1:2] = ['a', 'b', 'c']    
        [1, 'a', 'b', 'c', 4] 범위를 하면 원소 하나하나가 들어간다.
        a[1] = ['a', 'b', 'c']  
        [1, ['a', 'b', 'c'], 4] 원소를 지정하면 리스트 전체가 들어간다.  

        a[1:3] = [ ]  
        -> [1, 'c', 4] 공백 대괄호를 통해 해당 원소를 삭제할 수 있다.  
        del a[1] del객체로 삭제할 수 있다.  
    * 리스트 관련 함수들  
        1. append : 리스트에 요소 추가  
        a = [1, 2, 3]  
        a.append(4)  
        [1, 2, 3, 4]  
          
        a.append([5,6])  
        [1, 2, 3, 4, [5, 6]]  
        2. sort : 정렬  
        a = [1, 4, 3, 2]  
        a.sort()  
        [1, 2, 3, 4]  
        문자도 정렬 가능하다  
        
        3. reverse : 리스트의 순서를 그대로 거꾸로 뒤집는다
        a = ['a', 'c', 'b']  
        a.reverse()  
        ['b', 'c', 'a']  

        4. index : 값을 찾아 위치를 반환한다.  
        a = [1,2,3]  
        a.index(3)  
        2  
        a.index(1)

        5. insert : 리스트에 요소 삽입

        7. pop : 요소 끄집어내기  
        >>> a = [1,2,3]  
        a.pop()  
        3
        -> [1, 2]    
        a = [1,2,3]  
        a.pop(1)  
        2  
        ->[1, 3]  

        8. count : 리스트 내에 인자로 전달한 값이 몇 개 있는지  
        a = [1,2,3,1]  
        a.count(1)  
        2  

        9. extend : 인자로 리스트만 전달할 수 있으며 append처럼 리스트를 더한다

        a.extend([4,5])

    * 딕셔너리  
    연관배열 또는 해시라고 한다.  
    key와 그에 대응하는 value값을 쌍으로 가진다.  
    중괄호를 쓴다.  
    다음과 같은 형태를 가진다.  
    dic = {'name':'pey', 'phone':'0119993323', 'birth': '1118'}  
    value에는 문자열과 리스트도 넣을 수 있다.  

        1. 쌍 추가하기  
        a = {1: 'a'}  
        a[2] = 'b'  
        a  
        {2: 'b', 1: 'a'}  
        a['name'] = 'pey'  
        {'name':'pey', 2: 'b', 1: 'a'}  
        a[3] = [1,2,3]  
        {'name': 'pey', 3: [1, 2, 3], 2: 'b', 1: 'a'}  
        쌍이 추가되는데 규칙이 없다. 중요한 것은 '무엇이 추가되었는가 이다'  

        2. 요소 삭제  
        del a[1]    
        {'name': 'pey', 3: [1, 2, 3], 2: 'b'}  

        * 딕셔너리를 사용하는 방법  
        인덱스로 데이터를 찾는 튜플과 리스트와 달리 딕셔너리는 key값으로 value를 찾는다.  
        grade = {'pey': 10, 'julliet': 99}  
        grade['pey']  
        10  
        grade['julliet']  
        99  

        * 딕셔너리 만들 때 주의할 사항  
       -  key값이 중복되었을 경우 저장된 value중에 랜덤하게 값이 나온다. 그냥 key값은 중복해서 쓰지마라. 
       - key로 리스트는 쓸 수 없지만 튜플은 쓸 수 있다.  
       key값은 변하지 않는다는 속성을 가지고 있기 때문이다.  

       * key 리스트 만들기  
       a.key() : key리스트의 객체를 리턴한다.  
       list(a.key()) : key 리스트를 리턴한다.  

       * value 리스트 만들기  
       a.values() : value리스트의 객체를 리턴한다. 
       list(a.values()) : value 리스트를 리턴한다.  

       * key, value 쌍 객체 얻기  
       a.item() : key와 value의 쌍을 튜플로 묶어 객체로 돌려준다.  
    
       * key, value 모두 지우기  
       a.clear() : 딕셔너리 안의 모든 요소를 삭제한다.  

       * key로 value 얻기  
       a.get(' ') : key값을 넣으면 value를 리턴한다. 
       a[' ']로 접근해도 되지만 key가 없을경우 []는 에러를 띄고 get은 None을 리턴한다. (None은 '거짓' 을 의미)  

       * 해당 key가 딕셔너리에 있는지 조사  
       in : 해당 key가 있으면 True, 없으면 False 리턴  
       a = {'name':'pey', 'phone':'0119993323', 'birth': '1118'}
        'name' in a  
        True  
        'email' in a  
        False
